# Audio Asset Metadata Manager

A Python application for extracting, analyzing and organizing metadata from digital audio assets.

## Features

- MP3 metadata extraction
- WAV metadata extraction
- FLAC metadata extraction
- Audio asset analysis
- JSON export
- CSV export

## Technologies

- Python
- Mutagen
- Pandas

## Roadmap

- BPM Detection
- Key Detection
- Duplicate Detection
- Audio Library Organization
